# 🎮 게임 테스트 가이드

## 1단계: 초기 데이터 삽입 (최초 1회만)

터미널/PowerShell에서:

```bash
cd server
npm run init-game-data
```

출력 예시:
```
📝 게임 초기 데이터 삽입 시작...
✅ 한글 맞춤법 퍼즐 10개 삽입 완료
✅ 영어 단어 순서 퍼즐 10개 삽입 완료
🎮 게임 데이터 초기화 완료!
```

## 2단계: 서버 실행

**방법 1: 배치 파일 사용 (Windows)**
```
start-server.bat
```

**방법 2: 수동 실행**
```bash
cd server
npm start
```

서버가 정상 실행되면:
```
🚀 API server listening on http://0.0.0.0:5000
📱 Local access: http://localhost:5000
```

## 3단계: 클라이언트 실행

**새 터미널 창**에서:

**방법 1: 배치 파일 사용 (Windows)**
```
start-app.bat
```

**방법 2: 수동 실행**
```bash
cd client
npm start
```

클라이언트가 정상 실행되면 브라우저가 자동으로 열리고:
```
http://localhost:3000
```

## 4단계: 게임 접속

### 브라우저에서 접속:

1. **게임 페이지 직접 접속:**
   ```
   http://localhost:3000/game
   ```

2. **메인 페이지에서 이동:**
   - 홈페이지에서 "게임" 메뉴 클릭
   - 또는 URL에 `/game` 추가

## 5단계: 게임 테스트

### 한글 맞춤법 퀴즈 테스트:
1. "🇰🇷 한글 맞춤법" 탭 클릭
2. 문제 읽기
3. 정답 선택 후 "제출하기"
4. 정답 시 코인 +10개 획득 확인
5. "힌트 보기" 버튼으로 힌트 사용 테스트 (50코인 소모)

### 영어 단어 순서 맞추기 테스트:
1. "🇺🇸 영어 단어 순서" 탭 클릭
2. 한국어 문장 확인
3. 단어를 클릭해서 순서대로 배치
4. "제출하기" 클릭
5. 정답 확인 및 코인 획득

### 인앱 결제 테스트:
1. 하단 "🛒 상점" 섹션에서
2. "광고 제거" 버튼 클릭 → 구매 확인
3. "코인 1,000개" 등 구매 버튼 클릭
4. 코인 증가 확인

### 쿠팡 광고 테스트:
1. 퍼즐 완료 후 나타나는 광고 확인
2. "상품 보러가기" 버튼 클릭
3. 코인 보너스 받기 (10코인)

## 🔧 문제 해결

### 서버가 실행되지 않을 때:
```bash
cd server
npm install
npm start
```

### 클라이언트가 실행되지 않을 때:
```bash
cd client
npm install
npm start
```

### 데이터베이스 오류:
```bash
cd server
rm game.db  # 기존 DB 삭제 (선택사항)
npm run init-game-data  # 다시 초기화
```

### 포트 충돌:
- 서버 포트 5000이 사용 중이면: `server/index.js`에서 포트 변경
- 클라이언트 포트 3000이 사용 중이면: 자동으로 3001로 변경됨

## 📱 모바일 테스트

로컬 네트워크에서 모바일 기기로 접속:

1. PC와 모바일이 같은 WiFi에 연결
2. PC의 IP 주소 확인:
   ```
   Windows: ipconfig
   Mac/Linux: ifconfig
   ```
3. 모바일 브라우저에서 접속:
   ```
   http://[PC의_IP_주소]:3000/game
   ```
   예: `http://192.168.0.100:3000/game`

## 💡 개발 모드

계속 개발하면서 테스트하려면:

**서버 (자동 재시작):**
```bash
cd server
npm run dev  # nodemon 사용
```

**클라이언트 (Hot Reload):**
```bash
cd client
npm start  # 코드 변경 시 자동 새로고침
```
